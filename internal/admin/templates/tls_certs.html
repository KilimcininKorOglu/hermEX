TLS certificates

The certificate the gateway (front door) and the mail daemons present. An uploaded certificate overrides the config-file certificate and is picked up within about a minute — no restart. Leave the name blank for the default certificate, or set an SNI host name to serve a specific host. Private keys are stored but never shown again.
