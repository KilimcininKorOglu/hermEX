TLS certificates

The certificate the gateway (front door) and the mail daemons present. An uploaded certificate overrides the config-file certificate and is picked up within about a minute — no restart. Leave the name blank for the default certificate, or set an SNI host name to serve a specific host. Private keys are stored but never shown again.

Certificate mode

Manual serves the operator-uploaded certificates below. ACME has the gateway obtain and renew Let's Encrypt certificates automatically for every active domain — its apex and the mail/autodiscover/autoconfig hosts — over the TLS-ALPN-01 challenge, so each name must resolve to the gateway on :443. A mode change takes effect when the gateway restarts; certificate contents still hot-reload without one.
